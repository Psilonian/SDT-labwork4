## Описание проекта

Библиотека `geometric_lib` реализует базовые функции для работы с геометрическими фигурами:

- **круг** – вычисление площади и длины окружности;
- **квадрат** – вычисление площади и периметра.

В рамках лабораторной работы №4 реализованы **модульные тесты** (unit-тесты) для проверки корректности работы этих функций, в том числе на граничных и некорректных данных.

## Структура репозитория

Основные файлы:

- `circle.py` — функции для круга:
  - `area(r)` — площадь круга `пи * r * r`;
  - `perimeter(r)` — длина окружности `2 * пи * r`.
- `square.py` — функции для квадрата:
  - `area(a)` — площадь квадрата `a * a`;
  - `perimeter(a)` — периметр квадрата `4 * a`.

Файлы тестов:

- `test_circle.py` — unit-тесты для модуля `circle.py`;
- `test_square.py` — unit-тесты для модуля `square.py`.

Функции проверяются на:

- положительные целые и вещественные значения;
- граничные значения (0);
- некорректные значения (отрицательные числа — ожидается `ValueError`).

## Требования

- **Python 3.10+** (на macOS чаще всего запускается командой `python3`)
